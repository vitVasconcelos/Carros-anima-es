FP Selection é uma landing page interativa desenvolvida em **HTML, CSS e JavaScript**, projetada para exibir uma seleção de carros de forma moderna e dinâmica. A página apresenta um **carrossel de carros** com informações detalhadas, botões de navegação, indicadores de posição e animações suaves

## Funcionalidades
- Carrossel interativo de carros com **animações de entrada e saída**.  
- Navegação entre os carros usando **setas** (`prev` e `next`).  
- **Indicadores numéricos e de progresso** que mostram o carro ativo.  
## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica da página.  
- **CSS3**: Estilização e animações
- **JavaScript (Vanilla JS)**: Controle do carrossel e interação com indicadores.
